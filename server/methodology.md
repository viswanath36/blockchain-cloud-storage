# Methodology

## Step 1 — User Authentication
Users register and login using password and OTP.

## Step 2 — File Encryption
Files encrypted using AES before upload.

## Step 3 — Distributed Storage
Encrypted files stored on IPFS.

## Step 4 — Blockchain Recording
IPFS hash stored on Ethereum smart contract.

## Step 5 — Verification & Download
Hash verified from blockchain before file is returned.

---

## Development Model
Agile-based iterative development with testing after each module.
