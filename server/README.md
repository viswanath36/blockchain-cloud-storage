# Blockchain Integration with Cloud Storage for Secure and Transparent File Management

## 📌 Project Overview
This project implements a secure cloud storage system integrated with blockchain technology to provide tamper-proof, encrypted, and traceable file management.

Users upload files through a web interface. Files are encrypted, stored on IPFS, and their hashes are recorded on the Ethereum blockchain using a smart contract.

---

## 🎯 Objectives
- Secure file storage using AES encryption
- Transparent record keeping via blockchain
- Multi-factor authentication (Password + OTP)
- No MetaMask — server-managed blockchain wallet
- Role-based access control

---

## 🧩 Tech Stack
| Layer | Technology |
|------|-----------|
| Frontend | HTML, CSS, JS / React |
| Backend | Node.js, Express |
| Database | MongoDB |
| Encryption | AES-256 |
| Storage | IPFS |
| Blockchain | Ethereum + Hardhat |
| Auth | JWT + OTP |

## 🚀 Features
- User Registration & Login with MFA
